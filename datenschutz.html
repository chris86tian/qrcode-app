Erfassung von allgemeinen Daten und Informationen
Die Webapp erfasst keine personenbezogenen Daten direkt über Formulareingaben, die über die zur QR-Code-Generierung notwendigen temporären Daten hinausgehen. QR-Code Daten
Die von Ihnen zur Generierung eines QR-Codes eingegebenen Daten (URL, Kontaktdaten, WLAN-Daten, Text) werden ausschließlich zur Erstellung des QR-Codes verwendet und temporär auf dem Server verarbeitet. Hochgeladene Logos werden ebenfalls nur temporär zur Einbindung in den QR-Code verarbeitet. Es erfolgt keine dauerhafte Speicherung dieser Daten oder Logos auf unserem Server über den Generierungsprozess hinaus. Die generierten QR-Codes selbst werden nach kurzer Zeit (wenige Minuten) automatisch vom Server gelöscht.
